Space pour Jump

Le jeu se termine si on entre dans la zone de la pièce rose
On peut récolter les pièces, leur compte est affiché sur l'UI
Les escaliers fonctionnent avec une animation
La porte fonctionne avec de la physique
Une UI world rappelle les règles et les contrôles
Un menu permet de lancer le jeu, et on y revient quand le jeu se termine.
Les pièces produisent des particules (VFX) et la pièce rose est surmontée d'une point light
Le jump et la collecte de pièces produisent des sons
